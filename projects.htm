Bio-Inspired Sub-Carangiform Fish BOT



BIO-INPIRED SUB-CARANGIFORM FISH BOT
Summer Internship @eYSIP-2019
  • A BOT which performs basic actions done by a fish having Body Caudal Fin Propulsion(BCF) locomotion of the type sub-carangiform.
  • Basic actions inlude moving forward, left right and taking turns with variable speed.
  • Head of the body is 3D printed. Rest half is made using acrlyic sheets.
  • Arduino Nano is used for programming.
  • Uses Wi-Fi to communicated with the remote. NRF24L01 module is used.
  • Undulatory motion.

Ant BOT



ANT BOT
Theme @eYRC-2018
  • A BOT which performs basic actions done by ants such as collecting leaves, honey dews etc and removing trash from their hills.
  • Pick and place mechanism is made from scratch using measuring scales and hobby servo motors.
  • Arduino Nano and Raspberry Pi v3 is used.
  • Uses Pi-Cam to detect ArUco markers and blocks.
  • Path planning

Portable Braille



PORTABLE BRAILLE
Sangam @Pragyan-2019
  • A reading assistive device for visually impaired people.
  • Google's 'pytesseract' library is used for OCR.
  • Various feature such as- Speed control, Grade selection.
  • Supports both Grade 1 and Grade 2 braille.
  • Braille terminal is made using DOT MATRIX printer head.
